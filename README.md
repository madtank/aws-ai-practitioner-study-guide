## Exam Domains

1. Domain 1: AI/ML/Generative AI Fundamentals (28%)
   - Basic concepts and terminology
   - Types of machine learning
   - Generative AI concepts

2. Domain 2: AI/ML/Generative AI Business Problems (24%)
   - Problem formulation
   - Use case identification
   - Solution assessment

3. Domain 3: AI/ML/Generative AI Solutions and Architecture (28%)
   - AWS AI/ML services
   - Solution architectures
   - Integration patterns

4. Domain 4: AI/ML/Generative AI Implementation and Operations (20%)
   - Implementation best practices
   - Security and compliance
   - Monitoring and operations

## Study Tips

1. Focus on understanding core concepts rather than memorization
2. Practice with hands-on labs using AWS Free Tier
3. Take practice exams to identify knowledge gaps
4. Review AWS documentation for key services
5. Join AWS certification communities for study groups
